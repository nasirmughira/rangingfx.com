# Contributing

Thank you for your interest in contributing to the RangingFX Premium website.

## Code style

- Use TypeScript and Next.js App Router patterns.
- Keep Tailwind CSS classes organized and semantic.
- Run `npm run lint` before submitting changes.
- Format with `npm run format`.

## Branching and commits

- Use feature branches for new pages or enhancements.
- Write clear commit messages describing the change.
- Ensure changes are tested locally before pushing.
